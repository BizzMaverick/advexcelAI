How This Works:

This demo shows how the "highlight rows in red" functionality works in Excel AI Assistant:

  • The "Apply Red Highlighting" button adds the CSS class "red-row" to all data rows
  • The CSS class applies a light red background (#fef2f2) and red text color (#dc2626)
  • The header row remains unchanged
  • The "Remove Highlighting" button removes the CSS class

This is a simplified version of what happens when you use the "highlight rows in red" prompt in the main application.
